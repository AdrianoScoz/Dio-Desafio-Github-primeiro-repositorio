##Abra o Git bush 

git clone (mais o link)
cd ( mais respositorio *que é o nome da pasta onde está salvo)
Ctrl+L (Limpar)
git status (verificar o status e atualizar o mesmo)
git add . ou git add -a ( adcionar os arquivos )
git commit -m "o titulo que quer colocar" ( commit para github)
git push origin main ( enviar os dados para Github apos isso apenas aperta F5 )
